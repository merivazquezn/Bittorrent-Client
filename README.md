# 22C1-Polleria-Rustiseria

Bittorrent Client in Rust

## Prerequisites

2. install gtk4 library. For ubuntu:
```
  sudo apt-get install -y libgtk-3-dev
```
   


## Running

example for info:
```
RUST_LOG=info cargo run ubuntu.torrent
```

example for debug:
```
RUST_LOG=debug cargo run ubuntu.torrent
```
